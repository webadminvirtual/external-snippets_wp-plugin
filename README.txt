Pull in content from external sites to display in posts/pages.

== Description ==

Easily pull in content from external sites to display in posts/pages.  Adds a shortcode that you can pass a URL as well as a DIV ID that you want to target.  Also allows an option to fix all relative links except anchor links.

```
[fetch_external_content url="https://www.google.com" elementId="main" replaceurl=true]
```
